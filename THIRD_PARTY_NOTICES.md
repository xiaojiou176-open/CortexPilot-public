# Third-Party Notices

This file is the repository-level place for third-party notices that must be
stated outside package manifests or lockfiles.

## Current Scope

- dependency license details remain primarily in package manifests and lockfiles
- repository-level asset provenance is tracked in
  `configs/third_party_asset_registry.json`
- add explicit notices here when copied text, media, icons, or vendored source
  require repository-level attribution

## When To Update This File

Update this file when you add:

- vendored source code
- copied documentation excerpts
- third-party assets that require redistribution notice
- release bundles with extra attribution requirements

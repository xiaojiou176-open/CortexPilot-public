Ecosystem map

CortexPilot is a command tower for Codex and Claude Code workflows with a read-only MCP surface. This page exists to show where adjacent projects fit without turning them into fake hero claims.

Primary bindings

  • Codex: primary workflow audience
  • Claude Code: primary workflow audience
  • MCP: current protocol surface, read-only only

Adjacent ecosystem layer

  • OpenHands: broader agent-stack comparison and ecosystem mention
  • OpenCode: comparison-only and transition-sensitive
  • OpenClaw: different product category, kept out of the main front door

Why this matters

The goal is not to average every hot keyword into one hero line. The goal is to make CortexPilot easier to discover and easier to explain without pretending official partnerships, hosted status, or write-capable MCP.

Current guardrails

  • Do not present hosted operator as current product truth.
  • Do not present write-capable MCP as current product truth.
  • Do not elevate OpenHands, OpenCode, or OpenClaw into the main hero.

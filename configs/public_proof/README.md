# Public Proof Sources

This directory is the tracked source-of-truth for repo-owned public proof
ledgers, methodology notes, proof summaries, and machine-readable manifests.

The public docs tree should keep only:

- route pages under `docs/**/index.html`
- public media assets such as `svg`, `png`, and `gif`
- generated public machine outputs that those route pages intentionally expose

The files here are not route pages. They are tracked source material that
public routes and generators can read without leaving maintainer-grade
contracts inside `docs/`.
